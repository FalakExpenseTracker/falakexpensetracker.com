# Fixed Deposits
Fixed Deposits are money given to a bank, which pays interest on them so that your money may grow. This also comes under expense tracking, so we have it in this app. The Fixed Deposits screen is accesible from the main menu (Main Screen --> Main Menu --> Fixed Deposits). The screen is shown in fig 9.1. There is a button to add fixed deposits in the bottom right corner. It asks the rate of interest, principle and time. The app will calculate your fixed deposit as it increases with time, componded monthly.
